# AI Маркетинговый Ассистент

Маркетинговый ассистент на базе Claude 3.7, который помогает с маркетинговыми задачами.

## Установка

1. Убедитесь, что у вас установлен Python 3.8 или выше
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка

1. Создайте файл `.env` в корневой директории проекта
2. Добавьте ваш API ключ Anthropic в файл `.env`:
```
ANTHROPIC_API_KEY=ваш_ключ
```

## Использование

Запустите ассистента:
```bash
python marketing_assistant.py
```

### Основные функции:

- Ответы на маркетинговые вопросы
- Создание контента
- Анализ рынка
- Стратегическое планирование
- Настраиваемый системный промпт

### Изменение системного промпта

Вы можете изменить системный промпт ассистента, используя метод `set_system_prompt()`:

```python
assistant.set_system_prompt("Ваш новый промпт")
```

## Безопасность

- Никогда не публикуйте ваш API ключ
- Храните `.env` файл в `.gitignore`
- Регулярно обновляйте API ключ
